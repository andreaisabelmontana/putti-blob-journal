# Putti Blob Family Journal v2

Interactive emotional journaling web app where kids physically mold 3D blobs using physics simulation, assign colors/emotions, and create AI-generated art.

## Features

- Physics-based 3D blob molding with vertex dragging
- 8 emotions with color-coded interactions
- Daily journaling flow (2 minutes)
- 7-day history with physics replays
- Parent dashboard with mood analytics
- PWA support for mobile installation

## Tech Stack

- Three.js (3D rendering)
- CANNON.js (physics simulation)
- Vercel (deployment + serverless functions)
- Vanilla JavaScript

## Development

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Deployment

Automatic deployment via Vercel on push to main branch.
